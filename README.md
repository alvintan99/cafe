# Cafe Employee Management System

A full-stack application for managing cafes and their employees.

## Project Structure
- `/backend` - Node.js/Express backend
- `/frontend` - React frontend

## Quick Start

### Prerequisites
- Node.js >= 18
- PostgreSQL >= 14
- npm or yarn

### Development Setup
1. Clone the repository
2. Setup backend:
   ```bash
   cd backend
   cp .env.example .env
   npm install
   npm run dev
